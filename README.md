# BetterBot 2.0 🚀

BetterBot 2.0, Discord sunucunuzu yönetmek, güvence altına almak ve eğlendirmek için gelişmiş bir bottur.

## Özellikler
*   **Gelişmiş Moderasyon:** Sunucunuzu otomatik olarak koruyun.
*   **Canlı İstatistikler:** Sunucu ve kullanıcı verilerini anlık takip edin.
*   **Eğlence Komutları:** Topluluğunuzu eğlendirecek oyunlar ve aktiviteler.
